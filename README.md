### Spotify Streaming Analytics
**Problem:** Music streaming platforms need to understand what drives song popularity and optimize playlist curation

**Approach:** Statistical analysis of 114K tracks using correlation analysis, ANOVA, clustering, and hypothesis testing

**Impact:**
- üéµ Discovered "sweet spot" effect: tracks with danceability >0.7 AND energy >0.7 achieve 2.35x higher popularity (77.7 vs 33.3 avg)
- üìä Genre matters more than audio features: emo (48.1) outperforms reggaeton (23.9) by 100%
- üîç Individual features show minimal correlation with popularity (r<0.10), challenging assumption that "better sounding" = more popular
- üí° Explicit content shows modest 10.6% lift, likely confounded by genre

**Technical Achievement:** Rigorous statistical testing (Pearson, ANOVA F=132-373, t-tests, chi-square), professional visualizations, reproducible analysis
